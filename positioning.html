Fixed Position

(See below)
In fixed position, the element is removed from the normal document flow, and no space is created for the element in the page layout.

Static Position

In static position, the element is positioned according to the normal flow of the document. Default value.

Relative Position

One
In relative position, the element is offset a given amount from its normal position within the document, but without the offset affecting other elements.

Four

Relative Position

One
In absolute position, the element is taken out of the flow; thus, other elements are positioned as if it did not exist. The element is positioned relative to its nearest positioned ancestor (i.e., not static). In contrast, a relative positioned element remains in the normal flow of the document.
Three
Four

Sticky Position

In sticky position, the element is positioned according to the normal flow of the document, and then offset relative to its nearest scrolling ancestor and containing block. The offset does not affect the position of any other elements.
